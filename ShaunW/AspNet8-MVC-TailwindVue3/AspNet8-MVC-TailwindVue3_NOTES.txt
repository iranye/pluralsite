Install & Use tailwindcss:
--01. Install NPM Task Runner (VS Extension)
--02. Right-click on project => Open in Terminal (opens a PS CLI)
--03. node --version => [author = v18.17.0] [exeter = v18.20.5]
		--3a. update and use specific version:
		nvm install 18.17.0
		nvm use 18.17.0
--04. npm --version => [author = v9.6.7] [exeter = v10.8.2]
--05. npm init -y
--06. npx tailwindcss init
        npm i tailwindcss@3.4.17
--07. Add file in wwwroot: site.src.css with:
    @tailwind base;
    @tailwind components;
    @tailwind utilities;

--08. Update tailwind.config.js with:
    content: [
        "./Pages/**/*.{html,cshtml}",
        "./wwwroot/**/*.{html,htm}"
    ],
--09. Run tailwindcss command to generate site.css:
    npx tailwindcss -i ./wwwroot/site.src.css -o ./wwwroot/site.css
--10. update package.json to run tailwindcss command with watch-for-changes:
  "scripts": {
    "css": "npx tailwindcss -i ./wwwroot/site.src.css -o ./wwwroot/site.css -w"
  },
--11. test package.json update:
    npm run css
--12. Open Task Runner Explorer and right click "css" => Bindings => Project Open
